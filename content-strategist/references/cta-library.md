# CTA Library — MMIND.ai Content System

## CTA by Funnel Stage

### Top of Funnel (Awareness)
- "Folge mir für mehr KI-Praxis-Impulse."
- "Follow for weekly AI implementation insights."
- "Speichere diesen Post — du wirst ihn brauchen."
- "Save this post for later."
- "♻️ Teile diesen Post mit jemandem, der das hören muss."

### Middle of Funnel (Education / Methodology)
- "Welchen Schritt setzt du diese Woche um?"
- "Which step will you try first? Comment below."
- "Hast du ähnliche Erfahrungen gemacht? Erzähl mir davon."
- "Wie sieht das in deinem Unternehmen aus?"
- "Was ist dein grösstes Hindernis bei der KI-Einführung?"

### Bottom of Funnel (Conversion)
- "Klicke 'Let's talk' in meinem Profil für ein 30-Minuten-Gespräch."
- "Click 'Let's talk' on my profile for a 30-minute call."
- "DM mich mit [Keyword] für [Resource]."
- "Link zum [Resource] im ersten Kommentar."
- "Antworte mit '🔥' und ich schicke dir den Guide."

### Lead Magnet CTAs
- "Kommentiere '[Keyword]' und ich schicke dir den Link."
- "Link im ersten Kommentar 👇"
- "Kostenlos. Kein Gatekeeping."
- "Melde dich zum Flow Factor Newsletter an — jeden Freitag neue KI-Praxis."

## CTA Rules

1. **One CTA per post** — never confuse with multiple actions
2. **Match CTA to post type:**
   - Awareness → Follow / Save / Share
   - Education → Question / Save
   - Conversion → DM / Calendar / Link
   - Lead Magnet → Comment keyword / First comment link
3. **No external links in post body** (360Brew Rule) — links go in first comment
4. **Standard footer always comes after the specific CTA**

## First Comment Format

```
📎 Quellen / Sources:
→ [Source Name]: [URL]

[Optional: Teaser for newsletter or resource]
```

## CTA Performance Benchmarks (European SME)

| CTA Type | Expected CTR | Best For |
|----------|-------------|----------|
| Save/Bookmark | 2-4% | Education posts |
| Comment keyword | 1-3% | Lead magnets |
| Profile link click | 0.5-1.5% | Conversion posts |
| Share/Repost | 0.3-0.8% | Awareness posts |
| Newsletter signup | 0.5-2% | Lead magnets, Education |
